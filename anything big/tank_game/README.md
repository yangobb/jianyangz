# 坦克大战游戏

一个基于Pygame开发的简单坦克大战游戏。

## 功能特点

- 玩家控制绿色坦克
- 敌方红色坦克自动移动和射击
- 可摧毁的墙壁
- 计分系统
- 生命值系统
- 胜利/失败条件

## 操作说明

- 方向键：控制坦克移动
- 空格键：射击

## 如何运行

1. 确保已安装Python 3和Pygame
2. 运行以下命令：
   ```
   python main.py
   ```

## 游戏规则

- 玩家有3条生命
- 击败所有敌方坦克获胜
- 被敌方子弹击中失去一条生命
- 生命值为0时游戏结束
- 击中敌方坦克获得100分
- 可以通过射击摧毁墙壁

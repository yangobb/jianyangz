# 主流程转化漏斗：国内 vs 海外近14天

## 口径说明

- 数据范围：2026-04-13 至 2026-04-26。
- 流量链路表：`dws.dws_path_ldbo_d`，过滤 `wrapper_name in ('携程','途家','去哪儿')`、`user_type = '用户'`。
- 国内/海外：按 `is_oversea = 0/1` 区分；支付订单补充按 `dws.dws_order.is_overseas = 0/1` 区分。
- list页：列表曝光行；detail页：`detail_uid is not null`；booking页：`booking_uid is not null`；order页：`order_uid is not null`。
- 下单：主口径为 `without_risk_order_num` 归因下单；同时补充支付成功且未取消订单数。
- UV 使用 `approx_count_distinct` 近似去重，PV、归因订单、间夜、GMV、支付订单为精确聚合。汇总 UV 为日 UV 累加，适合看近两周日流量规模，不等同 14 天去重用户数。

## 结论

- 海外 list_uv 约为国内的 2.52%（238,116 vs 9,438,122）。
- 海外 L2D_uv 为 61.65%，低于国内 75.36%。
- 海外 detail 到 booking 的 UV 转化为 13.77%，低于国内 20.70%。
- 海外 booking 到 order 页 UV 转化为 16.70%，低于国内 49.78%。
- 归因下单口径下，海外 List 到下单为 1.58%，国内为 8.85%；支付订单数海外 5,136，国内 1,110,489。
- `Order到下单` 超过 100% 是因为分母为 order 页 UV，分子为归因下单数，同一 UV 可对应多笔下单。
